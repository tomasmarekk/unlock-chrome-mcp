# Contributing

This repo is intentionally small and practical.

Useful contributions:

- verified setup notes for another MCP client,
- fixes for new `chrome-devtools-mcp` package versions,
- safer platform launchers,
- clearer troubleshooting steps,
- verification output from macOS/Linux variants.

Please keep security implications visible. Any change that makes browser control easier should also keep the dedicated-profile and local-debug-port warnings clear.
